# Maverick Bank

A React-based frontend banking application named Maverick Bank with routing, authentication, and reusable components.

This project is a **React** single-page application built with **Vite** and **React Router**.

## Getting started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Then open the URL that Vite prints in the terminal (by default `http://localhost:5173/`) in your browser. The app will automatically reload when you edit source files.

## Available scripts

- `npm run dev` – start the Vite dev server
- `npm run build` – build the app for production into the `dist/` folder
- `npm run preview` – preview the production build locally

## Tech stack

- **React** (`react`, `react-dom`)
- **React Router** (`react-router-dom`)
- **Vite** (bundler/dev server)

There is no Angular code in this project; any previous Angular CLI references have been removed from this README to avoid confusion.
